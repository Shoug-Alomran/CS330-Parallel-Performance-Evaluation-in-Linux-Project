# CS330 Course Project

Welcome to the documentation site for our **CS330: Introduction to Operating Systems** course project. This site is used to organize the work across **two phases**, document the technical setup, and present the results and reports.

## Project at a Glance

This project focuses on operating system fundamentals in a Linux environment, with an emphasis on:

- Process management and process creation
- Parallel execution concepts
- Experimental performance evaluation
- Clear documentation and reporting

## Phases

### Phase I
Phase I documents the environment setup (virtual machine + Linux), building and running the provided program, and performance experimentation under different configurations.

### Phase II
Phase II will extend the project based on the instructor’s specifications once released. This section will be updated accordingly.
